Abstract

This research presents a brain tumor classification workflow using the Orange image data mining tool. The pipeline includes InceptionV3 image embeddings, dimensionality reduction with PCA, and systematic evaluation of six ML algorithms using k-fold (k=3, 5, 10) cross-validation. The neural network model consistently achieves classification accuracy above 96% and AUC up to 0.997, reaching a peak of 99.0% accuracy with AUC 0.999 in 10-fold validation — a +7% improvement over prior work.

Methodology

Data Collection

Preprocessing & Augmentation

03
Feature Extraction
InceptionV3 CNN → 2048-dim vectors per scan. Combined with handcrafted texture, contrast, and mean intensity features.

04
PCA Dimensionality Reduction
Reduces 2048-dim to 100–150 components retaining 95% of variance. Eliminates redundancy and reduces computation time by 70%.

05

6 classifiers: Neural Network (Adam), SVM, Random Forest (n=100, max_depth=10), Logistic Regression, Naive Bayes, Decision Tree.
